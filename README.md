GDALWRAP
========

*C++11 GDAL wrapper*

* http://gdal.org
* http://www.openrobots.org/wiki


INSTALL
-------

    git clone https://github.com/pierriko/gdalwrap.git && cd gdalwrap
    mkdir build && cd build
    cmake -DCMAKE_INSTALL_PREFIX=$HOME/devel ..
    make -j8 && make install
